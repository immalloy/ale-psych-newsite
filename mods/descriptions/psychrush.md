# Psych Rush

## Overview
Psych Rush is a fast-paced original mod that blends rhythmic combat with story-driven missions. Players dash through surreal arenas, build momentum, and chain combos to unlock new stages and rewards. The mod focuses on responsive input, a clean UI, and scalable difficulty so casual and competitive players can enjoy the ride.

**Highlights**
- Momentum-based combat flow
- Modular arenas and remixable challenges
- Built-in support for Windows and Linux
