# Solución de problemas de instalación

Usa esta página para resolver problemas comunes de configuración y lanzamiento.

## Dependencias faltantes

- Confirma que Haxe esté instalado y disponible en tu PATH.
- Vuelve a ejecutar el script de instalación de librerías de Haxe desde el repositorio.
- Asegúrate de que tu terminal pueda ejecutar `haxelib list` sin errores.

## Herramientas de compilación no encontradas

- Windows: instala Visual Studio 2022 con la carga de trabajo de C++ de escritorio.
- macOS: instala Xcode Command Line Tools.
- Linux: instala `build-essential` (o el equivalente de tu distro).

## Errores al iniciar

- Elimina compilaciones en caché y vuelve a ejecutar el comando de compilación.
- Elimina mods antiguos para comprobar un conflicto específico de un mod.
- Revisa la consola por el primer mensaje de error; normalmente apunta al asset o script faltante.
