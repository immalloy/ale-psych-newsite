# Instalar mods

Los mods se instalan colocándolos en la carpeta `mods/` y activándolos desde el menú Mods dentro del juego.

## Encuentra la carpeta de mods

- El motor busca un directorio `mods/` junto al ejecutable del juego.
- Si no existe, créalo manualmente y reinicia el juego.

## Instalar un paquete de mod

1. Extrae el zip o la carpeta del mod dentro de `mods/`.
2. Confirma que la carpeta del mod contenga archivos como `data.json`, `pack.json` o subcarpetas de assets como `images/` y `songs/`.
3. Evita carpetas anidadas (por ejemplo `mods/MiMod/MiMod/`).

## Activar el mod

- Abre el menú Mods con `Ctrl + Shift + M`.
- Activa el mod y reinicia si se te solicita.
