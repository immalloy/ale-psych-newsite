# Primer inicio y atajos

Esta página cubre la primera ejecución y los atajos de teclado predeterminados que puedes usar mientras pruebas una compilación. La mayoría de los atajos se pueden remapear en el menú de Opciones o en `Controls.json`.

## Lista rápida del primer inicio

- Confirma que la pantalla de título cargue y que el audio se reproduzca.
- Abre el menú Mods para asegurar que tu instalación lea el directorio `mods/`.
- Activa el overlay de depuración para verificar que las herramientas de desarrollador estén disponibles.

## Atajos predeterminados

| Atajo | Contexto | Descripción |
| --- | --- | --- |
| `Ctrl + Shift + M` | En cualquier parte excepto gameplay | Abre el menú de Mods para cambiar mods sin reiniciar. |
| `Ctrl + Shift + N` | En cualquier parte excepto gameplay | Reinicia completamente el motor, limpiando scripts, tweens y audio. |
| `F2` (Windows) | En cualquier parte | Abre la consola nativa de Windows para registros en vivo. |
| `F3` | En cualquier parte | Cambia modos del overlay de depuración (solo FPS, estadísticas completas, oculto). |
| `7` (modo desarrollador) | Gameplay | Abre el Editor de Charts cuando no está en pausa o en cinemáticas. |
| `8` (modo desarrollador) | Gameplay | Abre el Editor de Personajes. |
| `7` (modo desarrollador) | Menú principal | Abre el estado de editor maestro definido en `data.json`. |
| `R` / `F5` | Gameplay | Reinicia la canción actual. |

## Notas

- El overlay de depuración lee su diseño desde `debug.json` cuando lo provee un mod.
- El modo desarrollador debe estar habilitado para acceder a los atajos del editor.
