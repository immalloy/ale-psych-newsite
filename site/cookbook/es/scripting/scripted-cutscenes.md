# Cinemáticas con scripts

Las cinemáticas con scripts te permiten controlar animaciones, movimientos de cámara, diálogos y eventos temporizados antes o después de las canciones.

## Flujo de la cinemática

- Define un script de cinemática que se ejecute antes de iniciar la canción o después de terminarla.
- Usa timers o callbacks de eventos para secuenciar animaciones, diálogos y transiciones.

## Preparación de assets

- Precarga fondos, retratos de diálogo y audio para evitar tirones.
- Mantén los nombres de archivo consistentes con las referencias de tu script.

## Pruebas de reproducción

- Prueba en Modo Historia y Freeplay para confirmar que la cinemática se dispare.
- Usa los logs del modo desarrollador para confirmar el timing.
