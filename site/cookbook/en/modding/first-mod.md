# Your First Mod

Follow this guide to create a minimal mod you can enable in the Mods menu.

## Step 1: Create a folder

Create a new folder inside `mods/` using a short, lowercase name (for example `my-first-mod`).

## Step 2: Add metadata

Create a `pack.json` file in the mod folder. Include a name, description, and version so the Mods menu can display it.

## Step 3: Add a test asset

Add a small asset such as a new image in `images/` or a test song in `songs/` so you can confirm the mod loads.

## Step 4: Test in-game

1. Launch the game.
2. Open the Mods menu.
3. Enable your mod and restart if prompted.
