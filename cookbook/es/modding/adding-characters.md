# Agregar personajes

Los personajes se definen con assets de sprites más un archivo de datos que describe animaciones, offsets y metadatos.

## Assets de personajes

- Guarda los sprites de personajes en la carpeta `characters/` o `images/` de tu mod (según las convenciones de tu motor).
- Usa nombres consistentes para animaciones idle, sing y especiales.

## Datos del personaje

Crea una definición JSON del personaje que apunte al sprite sheet, nombres de animación y offsets. Mantén los valores consistentes con archivos de personajes existentes para que el editor pueda leerlos.

## Prueba el personaje

- Usa el Editor de Personajes (`8` en modo desarrollador) para cargar e inspeccionar tu nuevo personaje.
- Verifica loops idle, animaciones de canto y offsets en gameplay.
