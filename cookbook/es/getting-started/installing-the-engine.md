# Instalar el motor

Usa esta guía para configurar ALE Psych Engine en plataformas de escritorio. Los pasos siguientes se enfocan en obtener una compilación jugable para validar tu entorno antes de empezar a modear.

## Lo que necesitas

- Un sistema operativo compatible (Windows, macOS o Linux).
- Git para descargar el repositorio.
- Haxe y las librerías requeridas de Haxe.
- Herramientas de compilación de la plataforma (Visual Studio en Windows, herramientas de Xcode en macOS o GCC/Clang en Linux).

## Lista rápida de instalación

1. Clona el repositorio de ALE Psych.
2. Instala Haxe y las librerías requeridas.
3. Ejecuta los scripts de configuración del motor desde el repositorio.
4. Verifica que puedas iniciar el juego o compilar un objetivo de escritorio.

## Verifica la instalación

- Lanza el motor y confirma que llegues a la pantalla de título.
- Abre el overlay de depuración con `F3` para confirmar que las entradas funcionan.
- Revisa la consola o terminal por advertencias de dependencias faltantes.
